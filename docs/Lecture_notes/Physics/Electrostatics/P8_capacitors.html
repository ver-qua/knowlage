Конденсаторы

Определение

На практике часто используют такие приборы как конденсаторы. Конденсатор - устройство из двух проводников заряженных одинаковыми по модулю и разными по знаку зарядами \(q_1 = -q_2 = q\). Обычно между проводниками диэлектрик. Используют конденсаторы для того, чтобы накапливать электрический заряд - электрическую энергию.

Рассмотрим такую систему \[ \begin{cases} \varphi_1 = \alpha_{11}q_1 + \alpha_{12}q_2 \\ \varphi_2 = \alpha_{21}q_1 + \alpha_{22}q_2 \end{cases} \]

Так как заряда равны по модулю, то: \[ \begin{cases} \varphi_1 = (\alpha_{11} - \alpha_{12})q \\ \varphi_2 = (\alpha_{21} - \alpha_{22})q \end{cases} \]

Вводят такое понятие, как ёмкость конденсатора: \[ C = \frac{q}{U} \]

Где \(U\) – напряжение между проводниками, \(U = - \Delta \varphi\) (тут периодически возникает путаница, потому что напряжение тоже иногда называют разность потенциалов)

Из нашей системы мы можем выразить напряжение \[ \varphi_1 - \varphi_2 = (\alpha_{11} -\alpha_{12} - \alpha_{21} + \alpha_{22})q \]

И отсюда, учитывая симметрию \[ U = (\alpha_{11} -\alpha_{22} - 2\alpha_{12})q \]

И тогда ёмкость через ёмкостные коэффициенты \[ C = \frac{1}{ \alpha_{11} -\alpha_{22} - 2\alpha_{12}} \]

То же самое можно провернуть и с ёмкостными коэффициентами.

Ёмкости самых частых симметрий

Плоский конденсатор

Пусть у нас есть две пластинки площадью \(S\) с поверхностным зарядом в простейшем варианте \(-\sigma, \sigma\) с расстоянием \(d\) между ними. Между такими пластинами получается практически однородное поле, и заряды по пластинам распределяются равномерно.

Напишем напряжённости для пластин \[ E_1 = \frac{\sigma}{2\varepsilon \varepsilon_0} \]

И суммарное от двух пластин поле будет равно \[ E_ = \frac{\sigma}{\varepsilon \varepsilon_0} \]

Выразим напряжение, так как напряжённость поля везде константа, его можно вынести за интеграл \[ U = -\Delta\varphi = \int_o^d{E dr} = {\frac{\sigma}{\varepsilon \varepsilon_0}}d = \frac{qd}{S\varepsilon \varepsilon_0} \]

Откуда ёмкость будет: \[ C = \frac{S\varepsilon \varepsilon_0}{d} \]

Сферический конденсатор

Пусть даны две сферы одна в другой с радиусами \(R_1\), \(R_2\) и общим центром и зарядами \(+q\), \(-q\)

Через теорему гаусса выразим напряжённость поля между обкладками

\[ \oint_S{\vec D d\vec S} = q \]

\[ D \cdot 4\pi r^2 = q \]

Откуда \[ E = \frac{q}{4\pi\varepsilon_0 \varepsilon r^2} = \frac{kq}{\varepsilon r^2} \]

Найдём напряжение через интеграл \[ U = \int_{R_1}^{R_2}{E dr} = \frac{kq}{\varepsilon}\left(\frac{1}{R_1} - \frac{1}{R_2}\right) \]

Тогда ёмкость:
\[ C = \frac{q}{U} = \frac{\varepsilon}{k\left(\frac{1}{R_1} - \frac{1}{R_2}\right)} = \frac{4\pi\varepsilon_0\varepsilon R_1 R_2}{R_2 - R_1} \]

Цилиндрический конденсатор

Пусть даны два полых цилиндра, один в другом с общей осью. Радиусы \(R_1\), \(R_2\) и заряды \(+q\), \(-q\). Длинна одинакова.

Выводить всё не будем, оно очевидно, пусть тот же, напряжённость -> интеграл -> напряжение -> ёмкость

\[ E = \frac{2k\tau}{\varepsilon r} \]

\[ U = \frac{2k\tau}{\varepsilon} \ln{\frac{R_2}{R_1}} \]

И ёмкость \[ C = \frac{q}{U} = \frac{l \varepsilon}{2k \ln{\frac{R_2}{R_1}}} \]

Но тут удобно использовать понятие погонной ёмкости как и погонного заряда \[ C_n = \frac{\tau}{U} = \frac{C}{l} = \frac{2\pi \varepsilon_0\varepsilon}{\ln{\frac{R_2}{R_1}}} \]

Но это всё просто и по-сути дело техники, рассмотрим что-нибудь ИНТЕРЕСНОЕ

Двупроводная линия

Пусть у нас есть два цилиндра, находящиеся на расстоянии \(l\) друг от друга с одинаковым радиусом \(R\). Учтём, что \(l\) сравнимо с \(R\). Вспомним метод зеркальных отображений. Цилиндры заряжены с линейной плотностью \(\tau\).

Вспомним вывод перераспределения заряда в незаряженном проводящем цилиндре(напомните мне кто-нибудь добавить про это статью). Инверсия \(x\) будет \[ \begin{flalign} \left(l - x \right) = R^2\\ x^2 - l x + R^2 = 0 \\ x = \frac{l +- \sqrt{l^2 - 4 R^2}}{2} \end{flalign} \]

И нужно взять \(x\) меньше \(R\) \[ x = \frac{l - \sqrt{l^2 - 4 R^2}}{2} \]

Теперь зная новое положение зарядов можно посчитать напряжение, проще всего его посчитать напряжение создаваемое одним зарядом на другом \[ U_1 = \int{E dr} = \int_{R-x}^{l-2x}{\frac{2k\tau}{r}} \]

Ноль у нас в левом заряде. Интегрируем \[ U_1 = 2k\tau \ln{\frac{l-2x}{R-x}} = 2k\tau \ln{\frac{2\sqrt{l^2-4R^2}}{2R-l+\sqrt{l^2-4R^2}}} \]

И если всё прибрать(домноженные на сопряжённое), выйдет \[ U_1 = 2k\tau \ln{\frac{2R+l+\sqrt{l^2+4R^2}}{2R}} \]

Второй цилиндр будет создавать такое же напряжение межу, поэтому суммарное напряжение \(U=2U_1\)

Первый цилиндр создаёт
\(\varphi_1 - \varphi_2 = U_1\)

Второй цилиндр создаёт
\(\varphi_2 - \varphi_1 = -U_1\)

Знаки противоположны и тогда \(2(\varphi_1 - \varphi_2) = 2U_1\). Такое есть в случае плоского конденсатора. Первая обкладка создаёт разность потенциалов и вторая такую же и в общем из-за суммирования получается напряжение в два раза. Найдём погонную ёмкость \[ C = \frac{\tau}{U} = \frac{\pi \varepsilon \varepsilon_0}{\ln{\frac{\ 2R+l+\sqrt{l^2+4R^2}}{2R} }} \]
